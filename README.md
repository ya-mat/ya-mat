# 松本安弘（まつもとやすひろ）

## 研究分野
- 積分方程式の高速解法．特に，境界要素法の高速直接解法を研究しています．
- 通信用トンネルの維持管理に関する研究．

## 経歴
- 2015年4月 京都大学大学院 情報学研究科 博士課程入学 在学中
- 2014年4月 民間企業にて研究業務に従事 現職
- 2014年3月 京都大学大学院 情報学研究科 修士課程修了
- 2012年3月 東北大学 工学部 卒業
